Install

One package.
Bring your own world.

Geomap deliberately ships no basemap. Pair it with @cublya/world-atlas, its sibling TopoJSON package, or feed it any TopoJSON or GeoJSON you already have.

Terminalnpm
npm install @cublya/geomap
# recommended basemap (the package ships none)
npm install @cublya/world-atlas
ReactES modules
import { GeoMap, prepareCountries } from "@cublya/geomap";
import world from "@cublya/world-atlas/countries-50m.json"
  with { type: "json" };

const countries = prepareCountries(world, { exclude: ["AQ"] });

<GeoMap
  preset="light"
  countries={{ data: countries, onSelect: setCountry }}
  fit="world"
/>

Components

The same props, flat or spherical.

GeoMap draws a projected flat map; GeoGlobe renders a drag-rotatable orthographic globe with inertia. Layers, cameras and themes are shared, so switching projection is a one-word change.

Flat map<GeoMap />

GeoMap

A projected SVG world with every layer the package knows:

  • countrieschoropleth fills, selection, hover
  • markersscreen-constant pins
  • routesgreat-circle arcs between stops
  • livemoving objects on one shared loop
  • childrenyour own SVG in projected space
Orthographic<GeoGlobe />

GeoGlobe

The same props render a globe. Drag rotates with inertia, and markers or objects on the far hemisphere cull automatically. 

<GeoGlobe
  countries={{ data: countries, fill }}
  markers={pins}
  routes={arcs}
  autoRotate={4}
/>

Undecided? GeoView flips between both behind a built-in map ⇄ globe toggle, keeping the geographic centre across the switch.

Theming

Three axes.
No CSS file.

Styling is preset × palette × outline: orthogonal props rather than a preset per combination. The default preset "none" paints nothing at all; opt in for a complete look with one prop.

Axis Values Default What it sets
presetcolour mode none · light · dark "none" A complete AA-contrast palette for ocean, land, states and controls
palettefill treatment filled · minimal "filled" Filled land, or hue-less line art with a transparent ocean
outlineborder behaviour line · gap · raised · none "line" Hairline borders, cut-paper gaps, or raised land with a soft shadow

Every preset value resolves through var(--geomap-*, fallback), so an ancestor stylesheet can retheme every map with plain CSS variables, and countries.pattern adds hatch and dot encodings that stay readable without colour.

Under the hood

Built from parts you can import.

The components are thin compositions over exported utilities (projections, great-circle math, tweens, cameras), so advanced composition never needs a fork.

01

Cameras

useMapCamera and useGlobeCamera expose fitTo, flyTo and focus, honour reduced-motion preferences, and are plain framework-free stores when you need them outside React.

02

ISO identity

prepareCountries resolves every feature against a bundled ISO 3166-1 table: get("US"), get("usa"), get("840") and get("United States of America") are the same country.

03

Accessibility

The SVG is role="img" and focusable: arrows pan or rotate, +/− zoom, Home resets, and every country carries a native <title>.

Source and license

MIT, and small on purpose.

Geomap is MIT-licensed and built to stay out of your way: no product data, no fetching, no bundled basemap and no border politics. patchFeatures is the explicit hook for editorial geometry.

It pairs with @cublya/world-atlas, the sibling package that redistributes Natural Earth as quantized TopoJSON with documented boundary views.
